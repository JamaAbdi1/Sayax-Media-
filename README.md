# 🌍Sayax Media News Online Webpage

## 📌 Project Description

This project is a simple news webpage built using **HTML and CSS**.
It focuses on presenting information about the **Middle East conflict**, including a short article, relevant image, external news links, audio, and a breaking news ticker.

---

## 🎯 Features

* 📰 News article about the Middle East conflict
* 🖼 Online image related to war/conflict
* 🔗 Direct links to trusted sources (BBC & Fox News)
* 🔊 Online audio playback
* 📺 Animated breaking news ticker (scrolling text)
* 🎨 Styled layout using CSS

---

## 🛠 Technologies Used

* HTML5
* CSS3

---

## 📂 How to Run

1. Download or copy the project code
2. Save it as `index.html`
3. Open the file in any web browser

## 📖 Project Explanation

This webpage demonstrates how HTML and CSS can be used to create a simple news platform.
It includes:

* Structured content (headings, paragraph, media)
* External resources (links, images, audio)
* CSS styling for layout and design
* Animation for a real-time news ticker effect
